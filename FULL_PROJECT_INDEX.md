## 🚀 Getting Started (3 Steps)

### Step 1: Clone & Setup (30 sec)
```bash
npm install
cd backend
pip install -r requirements.txt
cp .env.example .env
# Add OPENAI_API_KEY to .env
```

### Step 2: Initialize (30 sec)
```bash
cd backend
python init_db.py
```

### Step 3: Run (30 sec)
```bash
# Terminal 1
npm run dev

# Terminal 2
cd backend && python main.py
```

**Access**: http://localhost:5173
